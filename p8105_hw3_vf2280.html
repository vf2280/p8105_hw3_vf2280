Important variables are the weather station ID, date of observation, precipitation (mm), snowfall, snow depth (mm), maximum temperature, and minimum temperature.

Do some data cleaning!

ny_noaa %>%
  janitor::clean_names() %>%
   separate(
        date, 
        into = c('year', 'month', 'day'),
        convert = TRUE
      ) %>%
  mutate(
    tmax = as.numeric(tmax),
    tmin = as.numeric(tmin),
    tmax = tmax/10,
    tmin = tmin/10,
    prcp = prcp/10
  )
